# motorola_cebu_dump

Actual repository located at [motorola_cebu_dump](https://gitlab.com/mistersmee/cebu).

This is because, GitHub fails to import the repo from GitLab, and fails to accept pushes from local repo, while GitLab has no problem.

GitHub recommends repos be less than 1 GB in size, and strongly recommends that they be less than 5 GB in size.

GitLab only has a 10 GB hard limit for total repository size.

GitHub blocks pushes >= 2 GB, while GitLab blocks pushes >= 5 GB.
